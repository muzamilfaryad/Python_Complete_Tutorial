# Python Learning Journey - Complete Course with Projects

This is a complete course with projects.

Welcome to the Python learning journey! This repository contains a comprehensive collection of Python learning materials, from foundational concepts to advanced Django and REST API development.

## 📚 Project Structure

```
Python_Course/
├── Python_Concepts/
│   ├── Complete_Django_Course/
│   │   ├── Try-Django/
│   │   │   ├── DRF/ (Django REST Framework)
│   │   │   └── Ecommerce_Project/ (Full-stack E-commerce)
│   │   └── src/ (Django Web Application)
│   ├── Complete_Python_Concepts/ (Python fundamentals)
│   ├── Contact_Book_Cli/ (CLI Application)
│   └── pytest/ (Unit Testing)
```

## 🎯 What's Included

### 1. **Django Courses**
- **DRF**: Django REST Framework API implementation
- **Ecommerce_Project**: Full-stack e-commerce application (Django backend + React frontend)
- **src**: Traditional Django web application with multiple apps

### 2. **Python Fundamentals**
- Data structures (lists, dictionaries, sets, tuples)
- Functions and classes
- File handling
- Error handling
- And much more!

### 3. **Practical Projects**
- Contact Book CLI: Command-line contact management application
- Pytest: Unit testing examples and best practices

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Node.js (for frontend projects)
- pip (Python package manager)
- Virtual environment (venv or virtualenv)

### Setup
1. Clone this repository
2. Create a virtual environment
3. Install dependencies from respective project directories
4. Refer to individual README files in each project folder

## 🛠️ Tech Stack

- **Backend**: Django, Django REST Framework
- **Frontend**: React.js
- **Database**: SQLite (default), PostgreSQL (Ecommerce)
- **Testing**: Pytest, unittest
- **Package Management**: pip (Python), npm (Node.js)

## 📖 Learning Path

1. Start with Complete_Python_Concepts for fundamentals
2. Move to Contact_Book_Cli for practical application
3. Explore pytest for testing best practices
4. Dive into Django courses for web development
5. Build the Ecommerce_Project for full-stack experience
