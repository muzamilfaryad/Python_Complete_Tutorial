# Python Concepts - Week 01

Welcome to my Week 01 Python internship practice workspace. This folder contains beginner-to-intermediate command-line programs built to strengthen core Python fundamentals through hands-on exercises.

## Highlights

- Practical scripts focused on real beginner workflows
- Clean progression: basics -> conditions -> functions -> OOP -> inheritance
- Interactive console apps (calculators, converter, quiz, Mad Lib)
- Early object-oriented design with reusable classes

## Learning Goals Covered

- Python syntax and program structure
- Variables, strings, numbers, and type conversion
- Input/output and user interaction
- Conditionals and comparison/logical operators
- Functions and return values
- Classes, objects, methods, and attributes
- Inheritance and code reuse
- Lists and dictionaries

## Project Files

| File | Purpose |
|---|---|
| `Hello.py` | Intro script for print statements, variables, strings, math operations, built-in math functions, and user input. |
| `Mad_Lib.py` | Small interactive Mad Lib program using multiple user prompts. |
| `calculator.py` | Minimal 2-number addition calculator. |
| `Basic_Calculator.py` | Menu-based calculator supporting `+`, `-`, `*`, `/` with divide-by-zero guard. |
| `unit_converter.py` | Menu-driven converter for kg->g, km->m, and C->F. |
| `if_statements.py` | Extensive if/elif/else practice, boolean logic, max-of-three function, and operator-based calculator flow. |
| `lists.py` | Practice with list creation, indexing, updating, slicing, and common list methods. |
| `Dictionaries.py` | Demonstrates dictionary creation, key access, updates, and lookups. |
| `functions.py` | Basic function definitions, arguments, and return-value examples. |
| `classes_and_objects.py` | Defines the `Student` class with an intro method and honor-roll check. |
| `Student.py` | Creates and tests `Student` objects imported from `classes_and_objects.py`. |
| `Chef.py` | Base `Chef` class with cooking methods. |
| `ChineseChef.py` | Inherited class extending `Chef` with a fried rice method. |
| `Cheff.py` | Demonstrates object creation and method calls for both `Chef` and `ChineseChef`. |
| `Quiz_App.py` | Quiz app with a `Question` class, prompt list, and score calculation. |

## Quick Start

1. Open terminal in this folder.
2. Run any file:

```bash
python Hello.py
python Basic_Calculator.py
python Quiz_App.py
```
